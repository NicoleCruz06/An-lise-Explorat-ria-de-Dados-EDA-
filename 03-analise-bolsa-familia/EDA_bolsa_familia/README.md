# EDA: Análise Exploratória do Programa Bolsa Família

Bem-vindo(a) ao repositório deste projeto de **Análise Exploratória de Dados (EDA)**, focado na investigação de dados públicos do **Programa Bolsa Família**.

Como estudante de Ciência de Dados e futura gestora pública, utilizei a biblioteca **Pandas** para ir além dos números frios, buscando compreender a distribuição, o impacto socioeconômico e o comportamento das concessões do benefício em diferentes recortes regionais e demográficos. O objetivo principal aqui é transformar dados brutos governamentais em evidências claras para a análise de políticas públicas.

---

## Objetivos da Análise

* **Mapeamento Regional:** Identificar as regiões e municípios com maior concentração de beneficiários e entender as disparidades geográficas.
* **Perfil Socioeconômico:** Analisar os valores médios repassados e investigar se há padrões que correlacionem o valor do benefício a indicadores locais.
* **Qualidade dos Dados:** Avaliar a consistência do banco de dados (dados nulos, duplicados ou discrepantes) para garantir uma análise confiável.
* **Geração de Insights:** Levantar hipóteses sobre a eficiência da cobertura do programa com base em dados estatísticos reais.

---

## Tecnologias e Bibliotecas Utilizadas

* **Linguagem:** Python 🐍
* **Manipulação e Tratamento de Dados:** Pandas e NumPy
* **Visualização Gráfica:** Matplotlib e Seaborn
* **Ambiente de Trabalho:** Jupyter Notebook / Google Colab
* **Fonte dos Dados:** Portal da Transparência / Base dos Dados (Dados Públicos Governamentais)

---

## Conceitos de Pandas Praticados Neste Projeto

Para estruturar esta análise, apliquei técnicas fundamentais da biblioteca Pandas:
1. **Leitura e Parsing:** Extração de grandes volumes de dados (arquivos `.csv` ou `.json`) de forma performática.
2. **Limpeza e Conversão:** Tratamento de valores ausentes, filtragem de registros e conversão de colunas de texto para formatos numéricos e monetários.
3. **Agrupamentos e Agregações:** Uso intensivo de `.groupby()`, `.pivot_table()` e funções agregadoras (`.mean()`, `.sum()`, `.count()`) para consolidar os dados por estado ou município.
4. **Análise Estatística Rápida:** Aplicação do `.describe()` e análise de quartis para identificar *outliers* e distorções nos repasses.
