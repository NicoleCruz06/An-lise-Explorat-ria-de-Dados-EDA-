# Análise Exploratória de Dados (EDA)

Bem-vindo(a) à minha pasta de estudos e projetos focados em **Análise Exploratória de Dados**. 

A análise exploratória é o meu primeiro contato com qualquer conjunto de dados. Aqui, o meu objetivo principal é investigar os dados, descobrir padrões, identificar anomalias, testar hipóteses e checar suposições com a ajuda de estatística e representações gráficas.

Como o meu foco de carreira é a **Análise de Políticas Públicas**, busco direcionar os meus projetos de EDA para entender problemas sociais, econômicos e estruturais, transformando dados brutos em insights que facilitem a tomada de decisão.

## Principais Conceitos Praticados Aqui

Nos projetos contidos nesta pasta, busco aplicar e consolidar conhecimentos em:
1. **Limpeza de Dados:** Tratamento de valores ausentes (*NaN*), remoção de duplicatas e correção de tipos de variáveis.
2. **Estatística Descritiva:** Análise de média, mediana, moda, variância, desvio padrão e quartis.
3. **Distribuição e Correlação:** Identificação de *outliers* (dados discrepantes) e análise de como as variáveis se relacionam entre si.
4. **Storytelling com Dados:** Criação de gráficos claros, intuitivos e que contam uma história visual sobre o problema analisado.
