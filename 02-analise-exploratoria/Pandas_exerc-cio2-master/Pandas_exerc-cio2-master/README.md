# Projeto 2 de Análise Exploratória com Pandas
Análise Exploratória de Dados a partir da biblioteca Pandas

## Sobre o Projeto
O projeto visa desenvolver e ampliar os conceitos das ferramentas da biblioteca **Pandas** após o projeto inicial ter fundamentado a base de conhecimento em análise exploratória de dados. No projeto eu analiso os dados de uma emenda parlamentar brasileira.

Instituição: Centro Universitário de Brasília (CEUB)

Curso: Ciência de Dados e Machine Learning

Disciplina: Introdução a Ciência de Dados

Período: 1º Semestre / 2026

## Objetivos do projeto
A atividade constitúi um conjunto de 16 exercícios que tem por objetivo ampliar o conhecimento básico das competências de **Pandas** e aumentar a gama de comandos aprendidos por meio de uma análise profunda dos dados de um DataFrame de emendas parlamentares, disponibilizado pelo professor da matéria.

## Aprendizados 
Permitiu um grande ganho de conhecimento nas áreas lógicas do uso dos comandos de **Pandas** durante a análise exploratória além de fundamentar conhecimentos básicos como o uso do **GitBash** e **GitHub** e aprofundar o nível de codificação no **VScode**, firmando a compreensão das tarefas diárias de um cientista de dados.
