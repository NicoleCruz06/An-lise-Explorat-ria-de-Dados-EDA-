# Projeto de Análise Exploratória com Pandas
Análise Exploratória de Dados a partir da biblioteca **Pandas** 

## O Projeto
O projeto visa permitir um contato inicial dos alunos com as ferramentas da biblioteca **Pandas** na análise exploratória de dados de uma emenda parlamentar brasileira.

Instituição: Centro Universitário de Brasília (CEUB)

Curso: Ciência de Dados e Machine Learning

Disciplina: Introdução a Ciência de Dados

Período: 1º Semestre / 2026

## Objetivos do projeto
A atividade constitúi um conjunto de exercícios que tem por objetivo incentivar o estudante a aprender o básico das competências de **Pandas** e explorar seus principais comandos durante uma análise de dados de um DataFrame de emendas parlamentares disponibilizado pelo professor da matéria.

Permitiu um grande ganho de conhecimento nas áreas lógicas do uso dos comandos durate a análise exploratória além de conhecimentos básicos como o uso do **GitBash**, **GitHub** e **VScode** nas tarefas diárias de um cientista de dados.
